* `IBM PC 5150`
* 最小的要求内存时 16KB
* DOS 1.0 至少需要32KB


**原因**

* 引导扇区有 512 字节
* 引导程序使用的栈还需要512字节

> 引导程序总共需要 512 + 512 = 1KB

* 把这1KB的空间，放到32KB的最后，似乎是一个比较合理的选择
