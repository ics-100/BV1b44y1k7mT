### 内中断

> `0x000` - `0x3ff`  / `1KB` 中断向量表

**单个中断**

* 4B
* 16位偏移地址: 16位段地址


**重要中断**

* `0` : 除法异常
* `0x80` Linux 系统调用

```
; push eflags
; push cs
; push ip

int
```

```
; pop ip
; pop cs
; pop eflags

; interrupt return
iret
```
